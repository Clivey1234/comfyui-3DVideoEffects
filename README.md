# 3D Video effects for Comfyui
Convert videos to 3D Anaglyph, 3D Pop, Pulfrich, SBS VR


Install

CMD into your ComfyUI\custom_nodes folder
git clone https://github.com/Clivey1234/comfyui-3DVideoEffects.git
